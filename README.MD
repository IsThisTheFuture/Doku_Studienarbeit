Dies ist eine Vorlage für Projektberichte und Bachelorarbeiten.

Zunächst sollten in der Datei einstellungen.tex die individuellen eisntellungen vorgenommen werden. Hierzu zählt bspw. Vorname, Nachname, Titel, ...

Die Datei dokumentation.tex ist die Hauptdatei, die übersetzt wird. In bibliographie.bib werden die verwendeten Quellen angegeben. 
Im Ordner ads befindet sich Anhang (appendix), Abstract, Glossar und die Liste der Acronyme. In den weiteren Datein hier sollten keine Anpassungen notwendig sein. 
Bilder sollten im Ordner images gespeichert werden.
Der eigentliche Inhalt der Arbeit befindet sich im Ordner content. Hier gibt es bereits fünf Kapitel, diese entsprechen der Makrostruktur und daher sollten keine weiteren Kapitel notwendig sein. Bei Bedarf können aber auch weitere Kapitel nach dem vorgegeben Namensschema erstellt werden.
